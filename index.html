HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta elements contain information about the metadata of the webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is the language for the formatting of pages and elements of HTML usually.
  • The order goes from out to inside: margin, border, padding, content.
  • You can make many boxes as is for each element in a page to customize it.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • code .: open vscode
  • cd "name of folder": to enter the correct folder in the current directory, ".." to go back one
  • git pull origin main: Pull request to update from the servers where the "origin" is "main" as an example
  • git push origin feature/starter-code: Pushes a request from the computer to the servers to update it under the starter code section

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Javascript is a programming language used to interact with variables and helps add interactivity to HTML pages
